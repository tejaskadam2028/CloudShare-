# CloudShare – Full Stack File Sharing Platform

A full-stack file sharing application built using Spring Boot, React, MongoDB, Clerk Authentication and JWT.

## Features
- Authentication with Clerk
- JWT Authorization
- User Profile Management
- Webhook Integration
- MongoDB Storage
- Credit System
- File Sharing

## Tech Stack

### Frontend
- React
- Axios
- Clerk

### Backend
- Spring Boot
- MongoDB
- Spring Security
- JWT

## Project Structure

```text
cloudshareapi/   → Backend
cloudshareweb/   → Frontend
```
